# 钱包页面布局优化 实现计划

> **目标：** 将钱包页面从双栏网格布局调整为全宽垂直堆叠，添加资金板块独立为全宽横栏，订阅计划区域下移并按 API Plan / Coding Plan 左右分栏展示
> **架构：** 纯前端布局调整，仅涉及 CSS class 和 JSX 结构变更。不修改任何业务逻辑、API 调用或数据流。RechargeFormCard 和 SubscriptionPlansCard 均变为全宽，垂直排列。SubscriptionPlansCard 内部计划列表从纵向堆叠改为左右分栏响应式网格。
> **技术栈：** React 19, TypeScript, Tailwind CSS

---

## 文件结构

### 修改文件
- `web/default/src/features/wallet/index.tsx` — 移除双栏网格布局（`xl:grid-cols-[...]`），RechargeFormCard 和 SubscriptionPlansCard 改为全宽垂直堆叠；移除 `showSubscriptionPanel` 状态及 `onAvailabilityChange` 回调
- `web/default/src/features/wallet/components/subscription-plans-card.tsx` — 移除 `onAvailabilityChange` prop 及相关逻辑；计划列表从纵向堆叠改为 API Plan（左）/ Coding Plan（右）左右分栏响应式布局
