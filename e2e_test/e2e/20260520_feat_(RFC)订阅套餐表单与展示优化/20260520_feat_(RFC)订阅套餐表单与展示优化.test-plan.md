## 已知阻塞项

- **套餐编辑抽屉渲染崩溃**：点击"新建套餐"或"编辑"按钮时，前端抛出 Base UI Error #60，页面显示 500 错误。此为前端 bug，需先修复后才能执行表单相关测试场景（场景 1-5）。钱包页卡片展示和购买弹窗（场景 6-10）不受影响。
- **现有套餐为禁用状态**：数据库中唯一套餐（#1）状态为禁用，钱包页不会展示。需要先通过 API 创建并启用至少一个 API 套餐和一个 CodingPlan 套餐作为测试前置数据，或等抽屉 bug 修复后通过 UI 创建。

### 6. 套餐卡片分区域展示
#### 6.1 钱包页按类型分为 API Plan 和 Coding Plan 两个区域

- 钱包页展示 "Coding Plan" 区域标题

#### 6.2 某类型无套餐时对应区域不显示
**需求来源**: 4.3.4 规则1 - 如果某类型没有已启用的套餐，该区域不显示
**前置条件**: 仅存在 API 套餐（无 CodingPlan 套餐）
**验证点**:
- API Plan 区域可见
- Coding Plan 区域不可见（不显示空区域）

### 7. 套餐卡片差异化显示

**验证点**:
- CodingPlan 套餐卡片 benefits 列表中包含 "Per 5h allowance: X tokens"
- 包含 "Weekly allowance: X tokens"
- 不包含 "Total Quota"
